# Playwright Automation Framework (Python)

Educational automation project built with Playwright and Pytest using the Page Object Model (POM) pattern.

The project demonstrates basic framework structure, reusable page objects, environment configuration, and pytest fixtures integration.

---

## Tech Stack

- Python 3.14
- Playwright (sync API)
- Pytest
- python-dotenv

---

## Run Tests

Install dependencies:

pip install -r requirements.txt

playwright install

Run tests:

pytest

Run in headed mode:

pytest --headed

This project is currently under development and being extended with additional test scenarios and CI integration.
